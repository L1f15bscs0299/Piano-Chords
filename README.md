# Piano-Chords

https://ndaidong.github.io/piano-chords/      -----------> LIVE DEMO


Install 
git clone link 
cd piano-chords

// install node packages
npm install

// download libraries
npm run setup

// start
npm start

Build
// build
npm run build
Test
// test
// ensure that it was built with "npm run build"
npm test

// e2e test
// ensure that node process is running with "npm start"
// then in another tab
npm run test:e2e
